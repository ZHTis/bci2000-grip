# GripFlight assets

Recommended PNG files:

- `player.png`: transparent player image.
- `background.png`: full-screen background.
- `obstacle_top.png`: top obstacle image.
- `obstacle_bottom.png`: bottom obstacle image.

Set their paths through `PlayerImage`, `BackgroundImage`, `TopObstacleImage`,
and `BottomObstacleImage`. Images are visual only; collision boxes come from
the map CSV so artwork changes do not change task behavior.
